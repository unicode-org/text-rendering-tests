GPOS–1: Pair Positioning, Format 1

The font for this test case contains a ‘GPOS’ table whose ‘kern’ feature enables a lookup of type 2 for Pair Adjustment Positioning. The test lookup contains a PairPos format 1 subtable to adjust the spacing between certain letter pairs, for example Ą and J. When an OpenType implementation renders the test strings, it should adjust the letter spacing accoording to the data in this subtable. If you see any overlapping glyphs, it’s a sign that your OpenType implementation fails to correctly interpret GPOS type 2 lookups with PairPos format 1 subtables.
